About

Something about this website...

This site is as much about the game (development) experiments, as it is about designing the site itself. It's really just a playground for me, so it's not unlikely to see unfinished games or perpetually changing website design on here. For that reason, a 'beta' label for the entire website seemed appropriate. The site is hosted on GitHub, using their excellent GitHub Pages service. As I quite like messing around with raw html, markdown languages and a static site is good enough for me, GitHub Pages is perfect for my needs.

Besides messing around with projects on this site, I'm also a (mostly PC) gamer at heart. I usually go by the name Lafie, which is unfortunately unpronounceable in English, or in any other language for that matter. Regardless, if you see that name, or the avatar on the left, say 'hello', because it's probably me. Unless someone read this and is now impersonating me, in which case it probably isn't me...

The name Turvy Topsy is a play on the term topsy-turvy, which means upside-down, backwards or disordered. Applying the meaning of topsy-turvy to itself, got me to the name Turvy Topsy. This website used to be called Turvy Topsy Studio, but being a lone developer just screwing around on here, the Studio suffix didn't seem appropriate, so I dropped it. If you're still reading this... well, I don't know why anyone would read through this... but if you are, and you want to contact me, you can do so on Twitter @TurvyTopsyGames.

So pretty...

by
Lafie
on 2012-10-15

I've now recognized that this website is just a playing ground for me, so I'm going to treat it as such. This means the entire site gets a 'beta' tag, and anything can change at anytime. This includes the content, the design, or whatever else I decide to play around with. I'm quite happy with the design right now though, so it's not unlikely that it will remain for some time. I still don't have a logo though, which is also a fun project, so expect that next.

The site is still hosted on GitHub Pages, which works great for someone not afraid to write articles in raw html. I did run into a problem with GitHub Pages' version of Jekyll and Liquid. Since the Liquid version on GitHub is several versions behind, it doesn't yet support the split command, which I use to split articles when they are too long to fit on the front page. So unfortunately, I'm forced to generate the site using a local installation of Jekyll, and git push the generated site, instead of letting GitHub generate the site for me. A minor inconvenience, although I can now check the site locally, and it will look exactly the same when synced. That is a nice unforeseen advantage, really.

P.S. I've discovered spell-checking in Notepad++, which is great when writing articles in html, you get the syntax highlighting, as well as the spell-checker. The best of both worlds!

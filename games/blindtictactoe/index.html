Blind Tic-Tac-Toe

This experiment has players playing Tic-Tac-Toe, without full knowledge of the moves of the other player. The game can be played in any web-browser that supports JavaScript. It only supports local multiplayer, so both players have to use the same device to play.

Including the partial knowledge of each player in a game of Battleship should make Tic-Tac-Toe a bit more interesting. What if each player can only see the other player's move if she happens to click a square occupied by the opponent. Other than that, Blind Tic-Tac-Toe plays just like regular Tic-Tac-Toe.

This little experiment was both an exercise in programming a game in JavaScript, as well as an idea I had to make Tic-Tac-Toe more interesting. I think the JavaScript code is clean and performs well, which isn't difficult for an application of this scope. The game on the other hand, does not seem to achieve what it was build to do. The first player seems to have an even larger advantage than with regular Tic-Tac-Toe, when his moves are invisible to the second player. Oh well... you play, you learn.

A clean, empty website with just the game on it can be found here.

Requirements

  • JavaScript enabled web-browser

Instructions

  • Play like regular Tic-Tac-Toe
  • If you're playing with a mouse, don't forget to move your cursor after a move, so the other player can't guess your turn from the cursor position.

MambaJS is a JavaScript remake of a small classic game called Mamba. It features snake-like gameplay with a few interesting twists and can be played right in your browser window.

After playing DOS game Mamba (1989) for years, it seemed appropriate to create a browser-based remake as a first hobby project to start off this website. The game is build in JavaScript using the gameQuery library. It is an unfinished project and will remain a work in progress for several more months, until it becomes appealing to finish an already months old project. Fortunately, the game is already playable and simple enough to finish between other projects. So without further ado, click the link below to start playing MambaJS.

Play MambaJS! (in-browser)

Requirements

  • JavaScript enabled web-browser
  • Keyboard

Controls

  • WASD or arrow keys to control the mamba snake
  • P to pause the game
  • R to restart the game

Instructions

  • Avoid hitting the edges of the map.
  • Avoid hitting your old skin, which is left behind on the map
  • Grab red apples for points
  • Grab gold apples for a lot of points
  • Don't get yourself stuck
